## Clone and install

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/ruzbyte/valstore-mobile.git
    cd valstore-mobile
    ```
  </Step>
  <Step title="Install Flutter dependencies">
    ```bash
    flutter pub get
    ```

    This resolves all `pubspec.yaml` dependencies including the `workmanager` git dependency (fetched directly from `https://github.com/fluttercommunity/flutter_workmanager.git`).
  </Step>
  <Step title="Add Firebase configuration">
    Valstore requires a Firebase project. See [Firebase integration](/dev/firebase) for full setup instructions. At minimum, add `google-services.json` to `android/app/`:

    ```bash
    cp /path/to/google-services.json android/app/google-services.json
    ```

    Without this file the app will crash at startup with a Firebase initialisation error.
  </Step>
  <Step title="Connect a device or start an emulator">
    ```bash
    flutter devices          # list connected devices
    flutter emulators        # list available emulators
    flutter emulators --launch <emulator-id>
    ```
  </Step>
  <Step title="Run in debug mode">
    ```bash
    flutter run
    ```

    The app locks screen orientation to portrait (`DeviceOrientation.portraitUp` / `portraitDown`) on device. This restriction does not apply on web.
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Firebase initialisation crash on startup">
    Ensure `android/app/google-services.json` exists and the package name in the file matches `android/app/build.gradle`. Re-run `flutter pub get` after adding the file.
  </Accordion>
  <Accordion title="workmanager dependency fails to resolve">
    The `workmanager` package is pulled from a git source. Ensure you have internet access and git is configured. Run `flutter pub get --verbose` to see the fetch progress.
  </Accordion>
  <Accordion title="flutter_inappwebview build errors on Android">
    Ensure `minSdkVersion` in `android/app/build.gradle` is at least `19`. `flutter_inappwebview` requires API level 19+.
  </Accordion>
  <Accordion title="Notification icon not showing on Android">
    `NotificationService` uses `@mipmap/launcher_icon` as the notification icon. Ensure `flutter_launcher_icons` has been run (`flutter pub run flutter_launcher_icons`) to generate the launcher icon assets.
  </Accordion>
</AccordionGroup>
